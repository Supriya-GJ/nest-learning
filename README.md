# Learning Platform Backend

A backend learning project built with NestJS.

## Features

* JWT Authentication
* Role Based Access Control (RBAC)
* PostgreSQL with TypeORM
* Redis Caching
* File Uploads
* Background Jobs with BullMQ
* Logging & Monitoring
* Unit & E2E Testing
* Docker basics & Docker Compose

## Tech Stack

* NestJS
* TypeScript
* PostgreSQL
* Redis
* Docker

## Run Locally

```bash
npm install
npm run start:dev
```

## Run with Docker

```bash
docker compose up --build
```
